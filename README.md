# Diabetes_Prediction_Project_using_ML
Diabetes Prediction Project using Machine Learning

# Overview
This Project predicts whether a patient is diabetic or not using Machine Learning.

# Problem Statement
Diabetes is a common disease that affects millions of people. Early prediction helps in better treatment.

# Dataset Features
Pregnancies   
Glucose  
Blood Pressure  
Skin Thickness  
Insulin  
BMI  
Diabetes Pedigree Function  
Age  

# Technologies Used
Python  
Pandas  
Numpy  
Matplotlib  
Scikit-Learn  
Google Colab  
ML Algorithm Used  
Support Vector Machine  

# Workflow
Data Collection  
Data Preprocessing  
Train-Test Split  
Model Training  
Prediction  
Evaluation  

# Output
The model predicts:  
Diabetic or Non-Diabetic
